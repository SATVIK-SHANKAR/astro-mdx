### ðŸ§¬ Face Recognition with K-Nearest Neighbors (KNN)

Each captured face image is flattened into a NumPy array. When a new face is detected, its array is compared against the dataset using Euclidean distance. The `k` nearest vectors vote on the label.

**Steps:**

1. Face is detected and cropped  
2. Image is resized and flattened to 1D  
3. Distance to known vectors is computed  
4. Most common label among `k` closest neighbors is returned

- âœ… No model training required  
- ðŸ“ˆ Scales well for small/medium datasets
